## 变量
在perl中`$`是标量变量,`@`是数组变量,`@ARGV`是命令行参数数组,`@_`是调用的函数的变量数组.  
我们通过shift来读取一个数组的第一个值并且会消除第一个变量.而push和pop就像操作一个栈一样来操作一个数组.不仅如此我们还可以`sort @array`这样来排序数组.随机访问数组类似这样 `array[number]`.  
数组的成员可以是不同类型的,但是这里sort的时候都会强转成字符串,然后再比较,哪怕是一样的也会转成字符串比较.  
而如果我们通过类似于c的方式写下[a1,a2]时,我们创建了一个类似指针的数组引用,这里我们可以用变量来绑定这个数组引用,然后通过$var->[0]的方式来访问这些元素.

## 正则表达
perl中用/ /来包住的部分是正则匹配的部分,这里有一个特殊运算符=~,表示对左值进行右侧的正则操作.比如:  
```perl
$bin_str =~ s/^\s+|\s+$//g;
```
这里s告诉perl是在进行搜索替换操作,g告诉perl是在进行全局的搜索替换,^匹配到开头,$匹配到结尾,中间的|是或,也就是要么从开头的那部分匹配出来,要么从结尾匹配一个字符串.最后两个/中间是空说明要将匹配的字符换为空字符.而匹配字段\s是空白字符,+是一个或多个.  
所以这里总的意思是删除字符串前后的空白字符.  
## 文件
perl中文件是也是通过句柄来访问的.我们使用open关键字  
```perl
  open my $fh, '<', $filename or die "无法打开文件 '$filename': $!\n";
```
这里是一个典型的打开方式,我们的$filename中存储着要打开的文件名(路径),'<'说明了打开的方式,这里是读.  
!!! Note
	打开方式有: '<'读取,'>'写入,'>>'追加'  
	die 是exit(1) 并且会输出错误信息.
	
我们通过`<$fn>`来通过句柄来访问文件流,在标量上下文中,也就是 `$var=<$fn>`的格式,会读取一行,而用`@array = <$fn>`会直接将整个文件全部读取出来.一般还会在读取出的行执行`chomp $line`这样来清除掉行尾的换行.
